Html - hypertext markup language - язык гипертекстовой разметки

<h1> - <h6> - заголовок
<p> - параграф
<blockquote> - цитата

<b> - жирный текст (bold)
<i> - курсив (italic)
<del> - зачеркнутый текст
<sub> - | текст снизу (нижний индекс)
<sup> - | текст сверху (верхний индекс)

<div> - блочный контейнер
<span> - строчный контейнер

<br> - перенос строки
<hr> - вставляет разделительную линию (горизонтальную)

Аргументы тегов

<div> class="MyClass"></div>
в данном примере class - это аргумент, а "MyClass" - это значение аргумента

Аргументы

  • class - устанавливает класс для элемента
  • id - устанавливает id для элемента
  • href - (для тега a) устанавливает ссылку

CSS - cascading style sheets - каскадные таблицы стилей

Внутри тега style - можно писать CSS стили для документа

<style>
/*
Стили
*/
</style>

подключение в html отдельным файлом


<!-- Внутри тега head -->
<link rel="stylesheet" href="style.css">

Селектор - выражение которое указывает к какой группе элементов указываются стили

пример:
body - для всего тега body
p - для всех тегов p
.MyClass - для всех элементов с классом "MyClass"
#MyId - для всех элементов с id "MyId"

Свойство - ключевое слово указывающие "что" конкретно стилизовать в элементе

пример:
color - свойство отвечающие за цвет текста
font-size - размер текста

всем параграфам цвет текста установить красный цвет

p {
color: red;
}

Установка значения свойства color:

  1. Словом (red, green, blue, yellow...)
  2. в RGB формате: rgb(/*значение красного*/, /*значение зеленого*/, /*значение синего*/)
  3. HEX: #FFFFFF
    1. Работает также как и RGB, но значения указываются в 16-ричном виде (по два числа) FF - красного, FF - зеленого, FF синего. Так как установлены максимальные значения для каждого из 3-х цветов - #FFFFFF - черный цвет (более подробно этот момент разберем в следующем уроке)
